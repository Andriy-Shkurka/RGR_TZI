Файл script.js містить основну логіку шифрування та розшифрування.
Він взаємодіє з елементами сторінки (textarea, input, button) і виконує такі кроки:

Функція log(message)
Виводить текстові повідомлення про процес (наприклад, “початок шифрування”, “успішно завершено”) у блок “Консоль процесу” на сторінці.
Це допомагає користувачу бачити хід виконання операцій у реальному часі.

1.Обробник події для кнопки “Зашифрувати”

2.Зчитує текст і ключ із полів введення.

3.Використовує метод CryptoJS.AES.encrypt(text, key) для шифрування.

4.Виводить результат у поле “шифрований текст”.

5.Записує повідомлення у консоль.

Обробник події для кнопки “Розшифрувати”

1.Зчитує шифрований текст і ключ.

2.Використовує метод CryptoJS.AES.decrypt(cipher, key) для розшифрування.

3.Якщо ключ правильний — показує розшифрований текст, інакше повідомляє про помилку.

1.Виводить деталі процесу у консоль.

Безпека
Дані не надсилаються на сервер — усе відбувається локально у браузері, тому користувацький текст і ключ не зберігаються та не кешуються.

<script src="https://cdnjs.cloudflare.com/ajax/libs/crypto-js/4.1.1/crypto-js.min.js"></script>

Це підключення зовнішньої бібліотеки [CryptoJS], яка реалізує алгоритм AES (Advanced Encryption Standard).
Без неї JavaScript не зможе шифрувати/розшифровувати текст, оскільки AES — це складний криптографічний алгоритм, і його краще не реалізовувати вручну.
